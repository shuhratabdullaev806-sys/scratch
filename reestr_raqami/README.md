# Reestr Raqami - Excel Shakli

Bu PDF hujjatdan Excel formatiga o'tkazilgan **Reestr Raqami** shaklidir.

## Fayllar

- `reestr_raqami.xls` - Excel fayli (Microsoft Excel yoki LibreOffice Calc bilan ochiladi)

## Qanday ishlatish

1. `reestr_raqami.xls` faylini Microsoft Excel yoki LibreOffice Calc dasturida oching
2. Kerakli maydonlarni to'ldiring:
   - **Sana** - Hujjat sanasi
   - **Mabla'ğlarni oluvchilar nomi** - Qabul qiluvchi nomi
   - **Mabla'ğlarni oluvchilar manzillari** - Manzillar
   - **Bank ma'lumotlari** - Bank nomi va kod
   - **Jadval** - Har bir qator uchun:
     - T/r (Tartib raqami)
     - Tuzilgan sanasi
     - Tuzilgan hujjat nomi
     - Tuzilgan hisob-fakturalar
     - Summa
3. Jami summa avtomatik hisoblanadi (agar kerak bo'lsa, formula qo'shing)
4. Imzolar qismini to'ldiring
5. Faylni saqlang va chop eting

## Eslatma

Ushbu fayl Excel-ga mos HTML formatida yaratilgan. Uni Excel dasturida ochganingizda, `.xlsx` formatda saqlashingiz mumkin.
